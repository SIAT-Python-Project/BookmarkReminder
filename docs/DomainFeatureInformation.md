# Domain별 기능 설명

### User
- 로그인
- 로그아웃
- 회원 가입
- 패스워드 변경
- 사용자 세션 생성
- 사용자 권한 확인


### Category
- 사용자별 카테고리 조회
- 특정 카테고리 조회
- 카테고리 이름 중복 검증
- 카테고리 추가
- 카테고리 수정
- 카테고리 삭제


### Bookmark
- 특정 북마크 조회
- 북마크 추가(+카테고리 이름 중복 검증 및 카테고리 추가)
- 북마크 삭제


### Memo
- 메모 조회
- 메모 추가
- 메모 수정
- 메모 삭제


### BookmarkCategory
- 카테고리에 포함되지 않은 사용자의 북마크 조회
- 카테고리별 북마크 조회
- 카테고리 및 카테고리별 북마크 삭제
